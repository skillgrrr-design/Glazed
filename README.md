
Ivoryk

Addon para meteor client
- **Minecraft:** 1.21.4
- **Fabric Loader:** 0.16.7+
- **Meteor Client:** 1.21.4

## Instalación

1. Descarga Meteor Client desde https://meteorclient.com/archive
2. Descarga Ivoryk 

3. Coloca ambos en la carpeta `mods` de tu instalación Minecraft

4. Inicia con el perfil Fabric y abre Meteor Client (Right Shift)

## Módulos


### Combat
- **TriggerBot** - Ataque automático cuando miras a un jugador
- **AimAssist** - Asistencia de apuntado suave

### Movement


- **NoSlow** - Elimina la ralentización al comer

### Player
- **ClickAction** - Lanzamiento rápido de Enderpearls y botellas de XP (keybindeable)
- **AutoGG** - Mensaje automático al matar jugadores (configurable)
